# HELLO WORLD IN GO

## [install go lang](https://golang.org/doc/install)

## copy the project:

to copy the project locally :

```sh 
git clone https://github.com/Night4dead/helloWorld.git
```

## run the project:

to run :

```sh 
cd helloWorld
```
 
```sh
go run .
```
